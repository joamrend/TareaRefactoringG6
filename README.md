# Tarea Refactoring - G6

Este repositorio contiene el desarrollo de la Tarea de Refactorización, cuyo objetivo principal es mejorar la calidad del código existente mediante la identificación de code smells y la aplicación de distintas técnicas de refactoring, manteniendo el comportamiento original del sistema.

## Objetivos
- Reorganizar y mejorar el código fuente.
- Reducir complejidad y duplicidad.
- Facilitar el mantenimiento futuro.
- Trabajar colaborativamente usando control de versiones.
- Documentar los cambios realizados y sus beneficios.

## Descripción

Para esta actividad se seleccionaron repositorios públicos de GitHub con código Java.
Sobre ellos se realizaron las siguientes tareas:
- Identificación de malos olores de código (code smells).
- Análisis y argumentación del problema encontrado.
- Aplicación de técnicas de refactorización adecuadas.
- Documentación antes vs después.
- Reporte de los cambios sugeridos en los repositorios analizados.
- Todo el trabajo respeta el comportamiento original del software.
